Тоді матимемо капіталу:
\begin{center}
I. Сталого 4840 + 484 = 5324

Змінного 1210 + 121 = 1331

\begin{center}
II. Сталого 1760 + 55 + 121 = 1936

Змінного 880 + 27\sfrac{1}{2} + 60\sfrac{1}{2} = 968
\end{center}

\begin{center}

 \text{Разом:} \left.\begin{aligned}
        \text{I. }5324 с + 1331 v = 6655\\
        \text{II. }1936 с + \phantom{0}968 v = 2904
       \end{aligned}
 \right\}

\begin{center}

 \left.\begin{aligned}
        \text{I. }5324 с + 1331 v + 1331 m = 7986\\
        \text{II. }1936 с + \phantom{0}968 v + \phantom{0}968 m = 3872
       \end{aligned}
 \right\}

\end{center}

Повторюючи це обчислення й заокруглюючи дроби, матимемо наприкінці

\begin{center}
